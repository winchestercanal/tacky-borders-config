## 📍 Instalação

Antes de utilizar este arquivo, é necessário ter o Tacky Borders instalado em seu sistema:

### 🔗 Repositório oficial
https://github.com/lukeyou05/tacky-borders

---

Após instalar o Tacky Borders:

1. Abra o arquivo [config.yaml](https://github.com/winchestercanal/tacky-borders-config/blob/main/config.yaml) disponibilizado neste repositório
2. Copie todo o conteúdo do arquivo
3. Cole no arquivo localizado em:

```powershell
%USERPROFILE%\.config\tacky-borders\config.yaml
```
Ou seja:
```powershell
C:\Users\SEU_USUARIO\.config\tacky-borders\config.yaml
```
> [!TIP]
> `%USERPROFILE%` é uma variável de ambiente do Windows que aponta automaticamente para a pasta do seu usuário.  
> Você pode colar esse caminho diretamente na barra de endereços do Explorador de Arquivos para abrir a pasta rapidamente.
---
⚠️ Importante:
Ao substituir o conteúdo do `config.yaml`, suas configurações anteriores do Tacky Borders serão sobrescritas.
Caso queira manter um backup das suas configurações atuais, faça uma cópia do arquivo original antes de substituir.
